# Tri-axis ATI Testbed

本项目为三轴自动化测控系统的 Python 上位机程序，用于集成控制运动控制卡、ATI 六维力传感器、NI 数据采集卡等设备。

## 硬件与依赖说明

*   **运动控制**: 以太网运动控制卡 (默认 IP: `192.168.0.1`)，底层依赖 `GAS.dll`。
*   **力觉传感**: ATI 力传感器，包含自定义报文解析与受力解算。需配合标定文件存放于路径下。
*   **数据采集**: NI-DAQmx 数据采集卡，用于信号管理。

## 目录结构

*   `src/ati_testbed/`: 核心控制模块封装（包含运动控制底层 DLL 动态库）。
*   `examples/`: 常用测控任务流程与上位机调用示例（如单点闭环运动）。
*   `tests/`: 硬件外设连接、系统初化配置测试脚本。
*   `data/`: 传感器标定及系统配置文件存放路径。

## 使用说明

1. 硬件连接：检查所有设备上电正常。
2. 网络配置：将上位机网卡 IP 配置为控制卡同号段（例如 `192.168.0.200`）。
3. 运动示例：
   ```bash
   python examples/example.py
   ```
